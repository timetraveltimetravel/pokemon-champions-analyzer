# 포켓몬 챔피언스 위협 분석기 — 프로젝트 보고서

> **공개 웹**: https://timetraveltimetravel.github.io/pokemon-champions-analyzer/
> 백업(Claude 아티팩트): https://claude.ai/code/artifact/205455d9-4924-49cc-a6e7-ea345b6d8601
> 로컬 실행: `npm run dev` (localhost:5173)

## 1. 목적

포켓몬 챔피언스 랭크배틀(싱글, 레귤레이션 M-B) 중에 **두 번의 검색만으로** 다음 세 가지를 즉시 판단하기 위한 도구:

1. **스피드** — 상대가 최속/준속/무보정/스카프일 때 각각 누가 선공인가
2. **받는 데미지** — 상대의 채용률 높은 기술이 나를 몇 타에 쓰러뜨리는가
3. **주는 데미지** — 내 기술이 상대를 몇 타에 쓰러뜨리는가

## 2. 데이터 소스 (4개)

| 소스 | 내용 | 갱신 |
|---|---|---|
| Smogon 월별 통계 (`smogon.com/stats`) | 포켓몬별 사용률·기술/도구/특성/스프레드 채용률. `gen9championsbssregmb` 포맷, 레이팅 1760+ 기준 | 매월 |
| Showdown 팀빌더 테이블 | **챔피언스 공식 합법 데이터**: 로스터 317종, 종별 학습셋(전체 무브풀), 합법 도구 148개, 기술 밸런스 패치 25건 | 수시 |
| Showdown pokedex | 챔피언스 신규 메가 등 npm 패키지에 없는 종의 종족값·타입·특성 | 수시 |
| PokeAPI GraphQL | 한국어 이름(포켓몬/기술/도구/특성) + 한국어 설명 텍스트 | 고정적 |

모두 빌드 시점에 `npm run fetch-data`로 내려받아 `src/gen/*.json`으로 번들한다 (Smogon은 CORS를 막아 브라우저에서 직접 못 가져오기 때문).

## 3. 계산 원리

- **엔진**: Pokémon Showdown 공식 데미지 계산 라이브러리 `@smogon/calc` (9세대 공식).
- **챔피언스 SP → 본가 노력치 변환**: SP는 스탯당 0~32, 총 66. 공식 변환 `노력치 = 8×SP − 4 (SP≥1), 개체값 31`을 적용하면 레벨 50 실능이 인게임과 **정확히 1:1 일치**한다. 즉 SP 1 = 실능 +1.
- **챔피언스 전용 보정**:
  - 신규 메가 34종 등 누락 종은 Showdown pokedex 종족값을 calc의 `overrides` 옵션으로 주입
  - 기술 밸런스 패치 25건 반영 (예: 만나자마자 90→100, 앵커샷 80→90, 부리캐논 100→120, 그로우 풀타입화)
  - 킬가르도는 공격 시 블레이드폼(공·특공 140), 돌핀맨은 마이티폼(공 160) 종족값으로 계산
- **검증**: 수기 계산과 대조 완료 — 예: 한카리아스 미러전 역린 116.7~137.2% (자속 1.5 × 드래곤 2배), 라이츄(메가Y) 기합구슬 49.1~58.3%, 랭크 +2 시 정확히 2배.

## 4. 기능 목록

- 한글/영어 검색 (IME 안전 입력, 마침표·공백·어순·"메가○○" 접두어 허용)
- 포켓몬 선택 시 통계 1위 스프레드·도구·특성 자동 적용, 상세 설정으로 커스텀 (상대도 성격·SP 직접 입력 가능)
- 종족값 → 실능 순서 표기, 타입 상성(약점/반감/무효, 특성 무효 포함) 배지
- ⚡ 스피드 판정: 최속/준속/무보정/통계 1위/스카프 시나리오별 선공 여부
- 🛡️ 받는 데미지 / 🗡️ 주는 데미지: 기술별 데미지 %·확정/난수 타수, 채용률, 랭크(-6~+6) 보정
- 통계에 없는 상대는 챔피언스 공식 학습셋 전체로 분석 (풀보정 가정)
- 기술/성격/특성/도구 호버 툴팁 (한국어 설명), 특수 메커니즘 안내 (탈, 배틀스위치, 마이트체인지, 변환자재)
- 📖 가이드: SP 시스템, 데미지 계산식, 랭크 배수표, 18×18 타입 상성표

## 5. 알려진 한계

- 연속기(스케일샷 등)는 계산기 기본 타수 가정
- 급소, 날씨·필드, 스텔스록 등 설치기 데미지는 미반영
- 따라큐의 탈은 판정에 미포함 (실전에서는 표시보다 1타 더 버팀 — 안내 문구로 대체)
- 통계는 배포 시점 스냅샷 (현재 2026-06)

## 6. 유지보수

```bash
npm run fetch-data       # 최신 통계·로스터·학습셋·패치 갱신
npm run dev              # 로컬 확인 (파일 감시는 폴링 — OneDrive 대응)
npm run deploy-pages     # 공개 웹(GitHub Pages) 재배포
npm run build-artifact   # dist/artifact.html 생성 → Claude 아티팩트 재배포용
```

매월 Smogon 통계가 갱신되므로 `fetch-data` → 재배포만 하면 된다. 챔피언스에 포켓몬·기술이 추가돼도 같은 명령으로 자동 반영된다.
